# `docs` Layout

This directory stores concise human-facing documentation.

## Naming convention

- Markdown docs use lowercase kebab-case.
- Root-level web assets stay at the root when they are part of the docs site.

## Current files

- `scripts_index.md`: script inventory
- `index.html`, `i18n.js`: lightweight docs/site assets
- `superpowers/`: plans and specs subdirectories
